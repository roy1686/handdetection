# NexusAI: Hand Gesture Control Hub

NexusAI is a powerful, browser-based web application that leverages state-of-the-art computer vision to translate your hand gestures into interactive experiences. No extra hardware required—just your webcam and your browser.

## Features

1. **Hand Detection**: High-performance real-time hand tracking.
2. **Finger Counter**: Instantly count the number of fingers held up, complete with voice synthesis.
3. **Air Canvas**: Draw in mid-air using your index finger. Erase with a closed fist.
4. **Rock Paper Scissors**: Play against an AI that adapts to your strategies.
5. **Gesture Calculator**: Perform calculations using simple hand gestures.
6. **Screenshot Trigger**: Take a screenshot using a specific hand pose.
7. **Speed Tracker**: Track how fast you can move your hands.
8. **Hand Heatmap**: Visualize the areas where your hands spend the most time.
9. **Gesture Memory**: Test your memory by repeating sequences of gestures.
10. **Picture Puzzle**: Solve puzzles using hand controls.
11. **Tic-Tac-Toe**: Play the classic game with gestures.
12. **Catch Game**: Catch falling objects using your hands.

## Technology Stack

- **React 18**: Component-based architecture.
- **Tailwind CSS**: Modern utility-first styling for a premium feel.
- **MediaPipe AI**: 60FPS advanced hand tracking from Google.
- **Vite**: Lightning-fast builds and local development.

## Getting Started

To run this project locally:

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the application in your browser (usually `http://localhost:5173`).

## Design & UI/UX

This project heavily prioritizes a premium, sleek aesthetic utilizing dark modes, smooth gradients, and glassmorphism. It is fully responsive and designed to work gracefully on mobile devices.

## Privacy & Security

All AI processing happens locally in your browser using MediaPipe. Your camera feed is never recorded, saved, or sent to any server. NexusAI ensures total privacy.
